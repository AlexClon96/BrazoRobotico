## Descripcion
Control de un brazo robótico con 5 grados de libertad con arduino y una interfaz gráfica en java.




***
## Material

 1.- Brazo Robotico de la marca Steren K-680 con 5 grados de libertad(base, codo, hombro, muñeca y pinza).

 2.- Arduino Mega 2560

 3.- Display LCD 16x2

 4.- Push button.

 5.- Cables.

 6.- Buzzer.

 7.- 3 Puente H  L298N

 8.- Protoboard.

 9.- Laptop.

10.- Fuente de poder.

11.- Un objeto
